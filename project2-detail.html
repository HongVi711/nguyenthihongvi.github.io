Project Overview

A mobile app that helps users learn and review English vocabulary effectively through offline dictionary lookup, flashcards, and mini-games. It supports user authentication and data synchronization with Firebase for a personalized learning experience. Built on MVC pattern and GetX for state management.

Technologies Used:

Flutter Firebase Dart

  • Implemented real-time data synchronization with Firestore.
  • Developed listening, speaking, reading, and writing exercises to enhance language learning.
  • Integrated speech recognition for pronunciation assessment.
  • Build vocabulary games
